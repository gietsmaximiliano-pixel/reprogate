# Maintainer Guide

Use ReproGate to ask for structured evidence without accusing reporters of using AI.

Recommended flow:

1. Run `reprogate init`.
2. Review the generated issue template and workflow.
3. Ask contributors to include a fenced `reprogate` YAML block.
4. Use labels as triage hints, not final decisions.
5. Generate receipts for reports you validate locally.
6. Use `verify-safe-run` only after manual review.

Never execute commands from public issues automatically.
